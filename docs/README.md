# Nestについて調べたことを記録する

### NestJSはMVCやMVPフレームワークではない
***
Webアプリケーションのサーバーとしての機能を提供するために作られている。  
最初の定型ファイルにはModelもViewもない。  

Service同士のつながりとしてアプリケーションが構築される。  
Controllerへと実態に即した粒度で機能が提供される。
